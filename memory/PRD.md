# Divine Iris Healing - PRD (Updated Mar 20, 2026)

## What's Built

### Core Platform
- Full website clone with admin panel
- Multi-currency support (INR/AED/USD + 30+ display currencies)
- Stripe payments with local currency charging
- VPN detection (India strict, others silent)
- Email OTP verification
- GST invoicing for India payments

### Enrollment & Payments
- Single-page enrollment (participants + promo + OTP → pay)
- Cart checkout flow (merged promo + OTP)
- Cross-sell discounts (tier-aware, multi-target)
- Combo discounts (tiered rules with tracking codes)
- Group discounts with program selection
- INR pricing for NRI students (3 methods: email whitelist, invite links, promo codes)
- Auto-close programs when deadline passes

### Student Dashboard (Phase B) — NEW
- **Soul Garden** — Farmville-style homepage with growth stages, seed grid, garden points
- **Bhaad Portal** — Release negativity through cosmic vortex animation, transforms into positive affirmations
- **Soul Tribe** — Community feed with posts, images, emoji reactions, comments
- **Calendar View** — Monthly calendar with color-coded sessions
- **Daily Progress** — Emotion/mood tracking with extraordinary moments
- **Program Pause/Resume** — Student-initiated with admin toggle

### Admin Features
- Planning calendar in Upcoming Hub (collapsible, editable)
- FOMO rotating subtitles on homepage
- Enrollment urgency testimonials
- Cross-sell rules editor with multi-target support
- Combo discount tiers with program selection
- GST invoice settings (editable, enable/disable)
- INR Override management (whitelist, tokens, promo)
- Live profit impact calculator
