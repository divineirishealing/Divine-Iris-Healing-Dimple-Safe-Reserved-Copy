Hero Banner

Sidebar: Hero Banner

Control the full-screen hero section that visitors see first when they land on your website. This is the most impactful visual on the site.

  • Hero Title & Subtitle — Edit the main heading and tagline displayed over the hero
  • Background Video — Upload a background video (.mp4) that auto-plays behind the hero text. Use the video upload button to replace it
  • Call-to-Action Buttons — Configure up to 2 CTA buttons with custom labels and links (e.g., "Book a Session", "View Programs")
  • Overlay Settings — Adjust the dark overlay opacity so text remains readable over light/dark videos
Tip: Keep the hero video under 15MB for fast loading. Compress it before uploading. The title should be 3-6 words for maximum impact.

Page Headers

Sidebar: Page Headers

Customize the hero banners and header images shown at the top of each internal page (Programs page, Sessions page, Transformations page, etc.).

  • Per-Page Headers — Set a unique background image, title, and subtitle for each page's header banner

Programs

Sidebar: Programs

The heart of your business — create, edit, and manage all healing programs. Each program has extensive configuration options.

Basic Information

  • Title, Category, Description — The core identity of the program
  • Image — Upload a program banner image displayed on cards and the program page
  • Duration — e.g., "90 days", "21 Days". Shown on upcoming cards (if toggled on) and in receipt emails
  • Session Timing & Timezone — e.g., "9PM - 10PM", "GST (Dubai)". Automatically converted to the user's local timezone on the site and in emails

Visibility & Enrollment

  • Visible on Site — Toggle to show/hide the program entirely
  • Show in Upcoming — Mark the program to appear in the "Upcoming Programs" section on the homepage
  • Flagship Program — Enables duration tiers (e.g., 1 Month, 3 Months, Annual) with separate pricing per tier
  • Group Program — Tags the program as a group/workshop
  • Enrollment Status — Set to Open, Closed, or Coming Soon. Closed programs show a configurable badge (e.g., "Seats Full", "Registration Closed")

Card Display Controls

  • Show Duration on Card — Toggle whether the duration badge appears on the upcoming program card
  • Show Pricing on Card — Toggle price display on cards
  • Show Tiers on Card — Toggle tier labels on flagship program cards
  • Show on Program Page Header — Independently toggle Duration, Start Date, and Timing for the program's detail page
  • Exclusive Offer Badge — Add a red badge like "Limited Time Offer" or "Early Bird Special" that overlays the card

Pricing

Program Links (shown in emails & success page)

  • WhatsApp Workshop Group — Toggle + URL. Sent in confirmation email with a green button
  • WhatsApp Community Group — A second WhatsApp link for a broader community group
  • Zoom Link — Sent in emails as a blue "Join Zoom" button
  • Custom Link — Any other link with a custom label

Program Page Content

  • Content Sections — Each program page is composed of sections defined in the Page Headers template. Here, you fill in the Title, Subtitle, Body Content, and Image for each section specific to this program
Important: After editing any program, always click the "Save" button inside the form. Reordering and visibility toggles save automatically.

Fraud Detection

Sidebar: Fraud Detection

An automated fraud monitoring system that flags suspicious enrollments based on multiple signals.

  • Alert List — View flagged enrollments with severity levels: Critical, High, Medium, Low
  • Detection Signals — VPN/proxy detection, IP-country mismatch, phone number inconsistency, timezone mismatch, BIN (card) country mismatch
  • Review Actions — Mark alerts as Reviewed, Confirmed Fraud, or Legitimate
  • Email Blocklist — Add email addresses to a fraud blocklist to prevent them from getting INR pricing
  • Search & Filter — Filter by severity and status
How It Works: When a user claims to be in India (for INR pricing), the system cross-checks their IP address, browser timezone, phone number, and card BIN. If any signals don't match, a fraud alert is created and the user is automatically switched to AED/USD pricing.

Quick Reference

Enrollment Flow (What Happens When a Client Enrolls)

StepWhat Happens
1. Participant FormClient fills in booker info + participant details (name, age, gender, country, attendance mode)
2. Email VerificationClient receives a 6-digit OTP via email and verifies it
3. PaymentClient sees pricing (auto-detected currency based on IP) and proceeds to Stripe checkout or India manual payment
4. ConfirmationOn successful payment: UIDs generated, confirmation email sent to booker, notification emails to opted-in participants
5. CRM UpdateClient added to Client Garden automatically

Currency & Pricing Logic

  • INR — Only shown to users verified as being in India (IP + phone + timezone + no VPN all must match)
  • AED — Default for Gulf countries (UAE, Saudi, Qatar, Kuwait, Oman, Bahrain)
  • USD — Default for all other countries
  • Currency Lock — Once a session starts, the currency is locked and won't change even if the user enters a different country in forms

Timezone Handling

  • Program timings are stored in the timezone you set (e.g., "9PM - 10PM GST")
  • On the website, timings are automatically converted to the visitor's local timezone based on their IP address
  • In receipt emails, timings are converted to the booker's IP-detected timezone
  • Participant notification emails convert timing to each participant's country timezone
