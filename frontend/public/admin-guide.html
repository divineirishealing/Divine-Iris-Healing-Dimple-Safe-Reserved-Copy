Complete Admin Panel Guide — Everything you need to know, explained simply

How to Access

Go to yoursite.com/admin → Enter your username & password → You're in!

The left sidebar shows all tabs. Click any tab to manage that part of your website.

All Tabs Explained

1. Hero Banner
The big beautiful banner at the very top of your homepage
What you can change
  • Background Image — Upload a photo or paste a URL
  • Title Text — The main heading (e.g., "Divine Iris Healing")
  • Subtitle Text — The tagline below it (e.g., "ETERNAL HAPPINESS")
  • CTA Button — The button text and where it links to
  • Overlay — Darken/lighten the image so text is readable
When to use: When you want to change the very first thing visitors see — new photo, new tagline, seasonal update.
2. Homepage ⭐
MASTER CONTROL for your entire homepage — titles, fonts, ordering, content
What you can change for each section
  • Title — The section heading (e.g., "Upcoming Programs")
  • Subtitle — Text below the heading (e.g., "CHOOSE YOUR HAPPINESS")
  • Font Controls — For both title AND subtitle:
    • Font Family (Lato, Cinzel, Playfair Display, etc.)
    • Font Size (18px, 24px, 36px, etc.)
    • Font Color (click the color swatch to pick any color)
    • Bold on/off
    • Italic on/off
  • Visibility Toggle — Show or hide any section
  • Reorder Arrows — Move sections up or down on the page
"Edit Section Content" button

For About, Sponsor, and Newsletter sections — click this to expand and edit the actual content (images, body text, descriptions) right here. No need to go anywhere else!

Tip: This is the ONE place for all section titles, subtitles, and their fonts. This is your best friend tab!
3. Page Headers
The banner image at the top of sub-pages (/about, /programs, /services, /contact)
  • Background Image — The banner photo for each sub-page
  • Overlay Color & Opacity — Darken the image for readability
  • Height — How tall the banner area is
Not the same as Homepage! Homepage = main page sections. Page Headers = the decorative banner at the top of OTHER pages.
4. Programs
Create and manage your healing programs (Quad Layer Healing, AWRP, etc.)
  • Add New Program — Click "+ Add Program"
  • Title, Description, Image — What the program is
  • Duration & Dates — When it runs
  • Pricing — Set prices in INR, USD, AED for each tier
  • Enrollment Status — "Open", "Coming Soon", or "Closed"
  • Tiers — Different levels (Standard, Premium, VIP)
  • Visibility — Show/hide from website
When to use: Launching a new program, changing prices, opening/closing enrollment, or updating program details.
Remember: Program CARDS get content from here. The section TITLE ("Flagship Programs") and its font are in the Homepage tab.
5. Pricing Hub
Quick controls for price visibility and attendance modes across ALL programs & sessions
  • Show/Hide Pricing — Toggle price visibility for each item
  • Attendance Modes — Enable/disable Online, Offline, In-Person
When to use: Quick way to show/hide prices or switch modes without editing each program individually.
6. Upcoming Hub
Controls which programs appear in the "Upcoming Programs" homepage section
  • Select which programs are marked as "upcoming"
  • Control their display order
7. Sessions
Create and manage personal healing sessions (Ancestral Healing, Chakra Healing, etc.)
Session Content
  • Add New Session — Click "+ Add Session"
  • Name, Duration, Description — Session details
  • Pricing — INR, USD, AED prices
  • Type — Online, Remote, In-Person
  • Available Dates & Times — When it can be booked
  • Visibility — Show/hide from website
Element Visibility (top of tab)
  • Toggle which elements appear in session cards on the homepage: Name, Type, Duration, Price, Calendar, Time Slots, Book Button, etc.
8. Testimonials
Manage video/media testimonials in the "Transformations" section
  • Video URL — YouTube or video embed link
  • Client Name — Who gave the testimonial
  • Thumbnail — Preview image
  • Visibility & Order — Show/hide and reorder
9. Text Quotes
Manage the scrolling text testimonial quotes on the homepage
  • Quote Text — What the client said
  • Client Name — Who said it
  • Program/Session — Which service it's about
  • Visibility & Order — Show/hide and reorder
10. Stats
The statistics bar showing numbers like "500+ Lives Healed"
  • Number — The count (e.g., "500+")
  • Label — What it means (e.g., "Lives Healed")
  • Icon — Visual icon for the stat
When to use: When your numbers grow and you want to update them!
11. Header & Footer
Top navigation bar and bottom footer on EVERY page
  • Header: Logo, navigation menu items, links, order
  • Footer: Menu links, contact email, social media, copyright text
12. Enrollments
View all client enrollments and registrations
  • Client name, email, phone
  • Program/session enrolled, payment status, date
  • Search, filter, and export as Excel
13. Inbox
Contact form messages from website visitors
  • Read messages, see sender details, reply
14. Client Garden
Your CRM — a database of all clients with labels and notes
  • Labels — Tag clients (VIP, New, Returning)
  • Notes — Add personal notes about each client
  • Search — Find specific clients
15. Promotions
Create promotional banners and announcements
  • Title, description, active dates, visibility
16. Discounts & Loyalty
Discount codes and loyalty rewards
  • Create codes, set amounts (% or fixed), define loyalty rules
17. Exchange Rates
Currency conversion rates for INR, USD, AED
  • Update rates when they change significantly
18. Subscribers
People who signed up via newsletter
  • Email addresses, dates, total count
19. Global Styles
Site-wide design settings for the entire website
  • Default heading and body fonts
  • Primary, secondary, accent colors
Careful: Changes here affect EVERY page on your site.
20. Payments
Payment gateway settings
  • Payment methods, bank account details, checkout instructions
21. India Proofs
Verify bank transfer/UPI payment screenshots from Indian clients
  • View uploaded proofs, approve or reject
22. Fraud Detection
Security system flagging suspicious enrollments
  • Review alerts, check severity, manage email blocklist
When to use: Check regularly to review flagged transactions.
23. Receipt Template
Customize the receipt clients receive after payment
  • Logo, header/footer text, which fields appear
24. API Keys
Technical settings for third-party integrations
Warning: Only change these if you know what you're doing — wrong values can break payments and emails!

Golden Rules

1. Always click "Save Changes" after edits in Hero Banner, Homepage, Page Headers, Header & Footer, or Global Styles

2. The Homepage tab is your best friend — most homepage changes happen here

3. Programs and Sessions tabs = what exists. Homepage tab = how it looks (titles, fonts)

4. Click "View Site" (top-right corner) to preview your changes after saving
