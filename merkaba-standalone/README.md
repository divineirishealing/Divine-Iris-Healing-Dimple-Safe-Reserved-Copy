# Divine Merkaba (standalone HTML)

This folder is **not** part of the divineirishealing.com React app. It is **not** deployed by default.

## Use

1. Download `merkaba.html` (or clone this repo and open the file locally).
2. Double-click `merkaba.html` or open it in any browser — works **offline** (`file://`).
3. To host it yourself, upload only `merkaba.html` to any static host, or drop it on Netlify / Cloudflare Pages / your own server.

Your picture is stored in **localStorage** in that browser only; nothing is sent to Divine Iris servers.

The view includes a **churning mid-body energy field** (swirling, harmonizing motion inspired by classic Merkaba / Merkabah visualizations). That motion is a **meditation image only**—not a description of any physical change in the body.
